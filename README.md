Convertendo números decimais em dinheiro (R$):

```
//com R$
var f = atual.toLocaleString('pt-br',{style: 'currency', currency: 'BRL'}).replace('R$', 'R$ ');

//sem R$
var f2 = atual.toLocaleString('pt-br', {minimumFractionDigits: 2});
```
